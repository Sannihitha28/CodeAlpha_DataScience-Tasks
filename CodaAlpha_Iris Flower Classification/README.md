# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview
This project predicts the species of an iris flower using supervised machine learning techniques.

The model was trained using Logistic Regression and evaluated using accuracy score, confusion matrix, and classification report.

---

## 📊 Dataset Information
The dataset contains 150 samples of three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

Features used:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 🎯 Model Performance
- Accuracy: 100%
- Evaluated using:
  - Confusion Matrix
  - Classification Report

The dataset is well-structured and mostly linearly separable, which explains the high accuracy.
